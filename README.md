# House Price Prediction Project

## Previsão de Preços de Imóveis com Machine Learning

Projeto desenvolvido no curso Desenvolvimento de IA para Análise Preditiva - Módulo 1.

## Objetivo

Desenvolver um modelo de Machine Learning para prever o preço de venda de imóveis utilizando características físicas e localização.

Dataset utilizado:

kc_house_data.csv

Variável alvo:

price

## Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Google Colab
- GitHub

## Modelo

Algoritmo utilizado:

Regressão Linear

## Resultados

MAE: 126.904,31

RMSE: 212.049,37

R²: 0,70
